# Changelog

---

## Menu Management Improvements 9-11-2025

- Implemented a **"Remove Dish" button** to remove dishes from the Menu Display Screen.
- Menu items displayed for removal on the Chef's **Add Dish** screen are now **grouped and sorted by Category (Starter, Main, Dessert)** for improved clarity.
- Implemented functionality to **load and persist the entire menu list** using AsyncStorage.

---

## Menu Display Features 9-11-2025

- Added a display for the **average price per category** on the Menu Display screen.
- Implemented **category filtering** ("All", "Starter", "Main", "Dessert") on the Menu Display screen.

---

## Login and Validation 9-11-2025

- Enhanced validation feedback on the **Login** screen by adding an **on-screen error message** for invalid credentials, to correct previous bug (inactive error message).
